## Project Overview
This project analyzes food delivery data by combining:
- orders.csv (transactions)
- users.json (user info)
- restaurants.sql (restaurant info)

## Join Logic
- orders.user_id = users.user_id
- orders.restaurant_id = restaurants.restaurant_id
- Left join used

## Files
- orders.csv
- users.json
- restaurants.sql
- final_food_delivery_dataset.csv
- analysis.py

## Analysis Includes
- Gold vs Regular users
- City-wise revenue
- Cuisine-wise revenue
- Rating-based performance
- Quarterly trends
